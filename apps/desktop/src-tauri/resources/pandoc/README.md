# Pandoc Binary Setup

Place the pandoc binary for your platform here:

## Download Links

- **macOS arm64**: https://github.com/jgm/pandoc/releases/download/3.6.2/pandoc-3.6.2-arm64-macOS.zip
- **macOS x64**: https://github.com/jgm/pandoc/releases/download/3.6.2/pandoc-3.6.2-x86_64-macOS.zip
- **Windows x64**: https://github.com/jgm/pandoc/releases/download/3.6.2/pandoc-3.6.2-windows-x86_64.zip
- **Linux x64**: https://github.com/jgm/pandoc/releases/download/3.6.2/pandoc-3.6.2-linux-amd64.tar.gz

## Expected Files

After extraction, place in this directory:
- `pandoc` (macOS/Linux)
- `pandoc.exe` (Windows)

## Development

For development, you can also install pandoc globally and the app will use the bundled version in production.
