# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-28

### Added

- Initial release of MD-DOCX Converter
- Tauri 2.x desktop application with React frontend
- Markdown to DOCX conversion via Pandoc
- Drag-and-drop file upload
- Custom DOCX template support
- Modern, responsive UI with dark theme
- Cross-platform support (macOS, Windows, Linux)

### Technical

- Vite 7 + React 19 + TypeScript frontend
- Tauri 2 with Rust backend
- Bundled Pandoc binary for offline conversion
